Aktuelle und zukünftige Versionen vom Oracle JDK und OpenJDK

15.04.2018 von Oliver B. Fischer

Am Mittwoch, den 16. Mai 2018 lädt die Java Usergroup Berlin-Brandenburg zusammen mit der adesso AG zum Vortrag Aktuelle und zukünftige Versionen vom Oracle JDK & OpenJDK mit Wolfgang Weigend von Oracle Deutschland ein. Der Vortrag Aktuelle und zukünftige Versionen vom Oracle JDK & OpenJDK (Wolfgang Weigend): Mit dem neuen JDK-Enhancement-Proposal (JEP 322), wird die zeitbasierte Java-Release-Versionierung definiert, die auf dem neuen Java SE Version-String-Schema basiert und die Namensgebung für die kommenden Versionen der Java SE Plattform und vom JDK vorgibt.
